# Trabalhos da Disciplina de Inteligência Artificial

Este repositório reúne os trabalhos desenvolvidos ao longo da disciplina de **Inteligência Artificial**, contemplando atividades teóricas e práticas relacionadas a diferentes técnicas, algoritmos e problemas clássicos da área.

O objetivo deste repositório é organizar os materiais produzidos durante a disciplina, documentar as soluções implementadas e facilitar a consulta e o acompanhamento da evolução dos estudos realizados.
